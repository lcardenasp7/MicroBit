🚀 Introducción a los Programas Digitales

¿Qué es un programa digital?

Un programa digital es un conjunto de instrucciones ordenadas que le dice a una computadora o dispositivo qué hacer paso a paso para resolver un problema específico.

🔄 Modelo Entrada-Proceso-Salida (EPS)

Entrada: Datos que ingresan (sensores, botones)
Proceso: Lo que hace el programa con esos datos
Salida: Resultado que produce (LED, sonido, movimiento)

🤖 Instrucciones vs Algoritmos

Instrucciones humanas: "Enciende la luz cuando sea de noche"
Algoritmo: IF luz < 50 THEN LED = ON

📱 Ejemplo Cotidiano: Alarma del Celular

Entrada: Hora actual del sistema

Proceso: Comparar hora actual con hora programada

Salida: Sonar alarma si las horas coinciden

🧮 Algoritmos, Pseudocódigo y Diagramas de Flujo

Un algoritmo es una secuencia finita de pasos lógicos y ordenados que resuelven un problema específico. Es como una receta de cocina, pero para computadoras.

📝 ¿Qué es el Pseudocódigo?

Es una forma de escribir algoritmos usando palabras del lenguaje natural mezcladas con estructuras de programación. Es más fácil de entender que el código real.

Ventajas:
  • Fácil de leer y escribir
  • No requiere sintaxis específica
  • Ayuda a planificar antes de programar

🔄 Símbolos de Diagrama de Flujo

Óvalo: Inicio/Fin
Rectángulo: Proceso/Acción
Rombo: Decisión
Paralelogramo: Entrada/Salida
Flechas: Flujo de datos

💡 Ejemplo Práctico: Encender Lámpara con Sensor

Pseudocódigo:
INICIO LEER nivel_luz SI nivel_luz < 50 ENTONCES ENCENDER lámpara SINO APAGAR lámpara FIN SI FIN

🎯 Micro:bit como Herramienta de Programación

¿Qué es la tarjeta Micro:bit?

Es una pequeña computadora programable del tamaño de una tarjeta de crédito, diseñada especialmente para aprender programación y crear proyectos tecnológicos de manera divertida.

🔧 Partes Principales del Micro:bit

🔲 Pantalla LED

25 LEDs en matriz 5x5 para mostrar imágenes, números y texto

🔘 Botones A y B

Botones táctiles para interactuar con tus programas

🌡️ Sensores Internos

Temperatura, luz, movimiento (acelerómetro) y brújula

📡 Bluetooth

Comunicación inalámbrica con otros dispositivos

🔌 Pines GPIO

Conexiones para sensores y actuadores externos

💻 ¿Qué es MakeCode?

MakeCode es un entorno de programación visual y por bloques desarrollado por Microsoft. Te permite programar arrastrando y conectando bloques, como si fuera un rompecabezas digital.

Características:
  • Programación por bloques (visual)
  • También permite código JavaScript
  • Simulador integrado
  • Funciona desde el navegador web

🌟 Tu Primer Programa: Mostrar un Ícono

  1. Ve a makecode.microbit.org
  2. Crea un nuevo proyecto
  3. Arrastra el bloque "mostrar ícono" desde "Básico"
  4. Elige un ícono (corazón, cara feliz, etc.)
  5. Haz clic en "Descargar" para transferir a tu Micro:bit

🔌 Sensores y Actuadores

📡 ¿Qué es un Sensor?

Un sensor es un dispositivo que detecta y mide condiciones del entorno (como luz, temperatura, movimiento) y las convierte en señales que la computadora puede entender.

⚡ ¿Qué es un Actuador?

Un actuador es un dispositivo que recibe señales de la computadora y realiza una acción física en el mundo real (como encender una luz, mover un motor, emitir sonido).

📊 Sensores Comunes en Micro:bit

☀️ Sensor de Luz

Detecta la cantidad de luz ambiente (0-255). Útil para lámparas automáticas.

🌡️ Sensor de Temperatura

Mide la temperatura en grados Celsius. Perfecto para estaciones meteorológicas.

🏃 Acelerómetro

Detecta movimiento, inclinación y gestos. Ideal para juegos y alarmas.

🧭 Brújula

Detecta dirección magnética. Útil para navegación y orientación.

🔧 Actuadores Comunes

💡 LEDs

Emiten luz de diferentes colores. Perfectos para indicadores y decoraciones.

🔊 Buzzer/Altavoz

Emite sonidos y música. Ideal para alarmas y proyectos musicales.

🔄 Servomotor

Motor que se mueve a ángulos precisos. Útil para robots y mecanismos.

📱 Pantalla LCD

Muestra texto e información. Perfecta para mostrar datos detallados.

🚨 Ejemplo: Alarma de Movimiento

Sensor: Acelerómetro detecta movimiento

Proceso: Si hay movimiento fuerte, activar alarma

Actuador: LED parpadea y buzzer suena

Si acelerómetro > 1000: Mostrar ícono de alerta Reproducir tono de alarma Encender LED rojo

🚦 Proyecto 1: Semáforo Inteligente

Problema: Los semáforos tradicionales tienen tiempos fijos que no se adaptan al tráfico real, causando esperas innecesarias.

Solución: Un semáforo que detecta movimiento (peatones o vehículos) y ajusta sus tiempos dinámicamente.

📋 Análisis EPS del Semáforo

📥 ENTRADA

  • Sensor de movimiento (acelerómetro)
  • Botón de cruce peatonal
  • Sensor de luz (modo nocturno)
  • Tiempo transcurrido

⚙️ PROCESO

  • Evaluar estado actual del semáforo
  • Verificar si hay solicitud de cruce
  • Calcular tiempo de espera
  • Cambiar estado según condiciones

📤 SALIDA

  • LEDs: Rojo, Amarillo, Verde
  • Pantalla LED: Contador de tiempo
  • Buzzer: Señal sonora para invidentes
  • Display: Estado actual

🌱 Proyecto 2: Mini Invernadero Automatizado

Problema: Las plantas necesitan condiciones específicas de temperatura, humedad e iluminación para crecer, pero es difícil monitorear constantemente.

Solución: Un invernadero que mantiene automáticamente las condiciones ideales para el crecimiento de plantas.

💻 Código del Invernadero

// Mini Invernadero Automatizado let temperaturaIdeal = 25 // °C let humedadMinima = 40 // % // Pines de componentes let pinBombaAgua = DigitalPin.P1 let pinVentilador = DigitalPin.P2 // Función de riego automático function activarRiego(duracion: number) { basic.showIcon(IconNames.Umbrella) pins.digitalWritePin(pinBombaAgua, 1) basic.pause(duracion) pins.digitalWritePin(pinBombaAgua, 0) basic.showIcon(IconNames.Yes) } // Control manual de riego input.onButtonPressed(Button.A, function () { activarRiego(3000) // 3 segundos de riego }) // Bucle principal del invernadero basic.forever(function () { let temperatura = input.temperature() let humedad = Math.map(input.lightLevel(), 0, 255, 100, 0) basic.showNumber(temperatura) basic.pause(2000) // Control automático if (humedad < humedadMinima) { activarRiego(2000) } if (temperatura > temperaturaIdeal + 5) { pins.digitalWritePin(pinVentilador, 1) basic.showIcon(IconNames.Surprised) } else { pins.digitalWritePin(pinVentilador, 0) basic.showIcon(IconNames.Happy) } basic.pause(5000) }) basic.showString("INVERNADERO")

🌦️ Proyecto 3: Estación Meteorológica Personal

Problema: No hay información meteorológica específica del entorno escolar o del hogar.

Solución: Una estación que mide y registra condiciones climáticas locales en tiempo real.

💻 Código de la Estación

// Estación Meteorológica let temperaturas: number[] = [] let modoDisplay = "temperatura" // Función para leer sensores function leerSensores() { return { temperatura: input.temperature(), luz: input.lightLevel(), humedad: Math.map(input.lightLevel(), 0, 255, 80, 30) } } // Cambiar modo de visualización input.onButtonPressed(Button.A, function () { if (modoDisplay == "temperatura") { modoDisplay = "luz" } else { modoDisplay = "temperatura" } basic.showString(modoDisplay) }) // Bucle principal basic.forever(function () { let datos = leerSensores() // Almacenar temperatura temperaturas.push(datos.temperatura) if (temperaturas.length > 10) { temperaturas.shift() } // Mostrar según modo if (modoDisplay == "temperatura") { basic.showNumber(datos.temperatura) } else { basic.showNumber(datos.luz) } // Alertas if (datos.temperatura > 30) { basic.showIcon(IconNames.Sad) } else { basic.showIcon(IconNames.Happy) } basic.pause(3000) }) basic.showString("METEO")

🎨 Diseño de Proyectos Tecnológicos

Diseñar un buen proyecto tecnológico requiere seguir un proceso ordenado que comienza identificando un problema real y termina con una solución funcional y bien documentada.

1

🔍 Identificación del Problema

Observa tu entorno escolar: ¿qué problemas puedes resolver con tecnología?

Ejemplos:
  • Luces que se quedan encendidas
  • Temperatura incómoda en aulas
  • Olvido de materiales
2

💡 Planteamiento de Solución

Define cómo Micro:bit puede ayudar a resolver el problema identificado.

Pregúntate:
  • ¿Qué sensores necesito?
  • ¿Qué acciones debe realizar?
  • ¿Cómo interactuará el usuario?
3

✏️ Bocetos y Estructura

Diseña la estructura física de tu proyecto usando materiales disponibles.

Materiales sugeridos:
  • Cartón y cartulina
  • Materiales reciclados
  • Piezas de Lego

⚙️ Programación del Prototipo

🔀 Condicionales (if/else)

Permiten que el programa tome decisiones basadas en condiciones.

Si botón A presionado: Mostrar corazón Sino: Mostrar cara triste

🔄 Ciclos (forever/repeat)

Forever: Ejecuta código continuamente
Repeat: Ejecuta código un número específico de veces

Para siempre: Leer temperatura Mostrar en pantalla
🎯 Tip de Programación: Siempre prueba tu código en el simulador de MakeCode antes de transferirlo al Micro:bit real.

La documentación es la memoria de tu proyecto. Te permite recordar cómo funciona, compartirlo con otros, y mejorarlo en el futuro.

🧠 Principios Básicos de Inteligencia Artificial

¿Qué es la Inteligencia Artificial?

La IA es la capacidad de las máquinas para realizar tareas que normalmente requieren inteligencia humana, como reconocer patrones y tomar decisiones.
